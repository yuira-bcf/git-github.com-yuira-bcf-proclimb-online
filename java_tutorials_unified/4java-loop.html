繰り返しと制御文の組み合わせ

📑 この章の内容

  • 4-1whileによる繰り返し
  • 4-2forによる繰り返し
  • 4-3制御文の組み合わせ
  • 4-4繰り返しの制御
Section 4-1

whileによる繰り返し

1.1 while文

条件がtrueの間、処理を繰り返します。

while文 Java 
int i = 0;
while (i < 5) {
    System.out.println(i);  // 0, 1, 2, 3, 4
    i++;
}

1.2 do-while文

最低1回は処理を実行し、その後条件を判定します。

do-while文 Java 
int i = 0;
do {
    System.out.println(i);  // 0, 1, 2, 3, 4
    i++;
} while (i < 5);

// 条件がfalseでも1回は実行
int j = 10;
do {
    System.out.println(j);  // 10（1回は実行）
} while (j < 5);
Section 4-2

forによる繰り返し

2.1 for文

基本的なfor文 Java 
// 構文: for (初期化; 条件; 更新) { 処理 }
for (int i = 0; i < 5; i++) {
    System.out.println(i);  // 0, 1, 2, 3, 4
}

// 複数の変数を使用
for (int i = 0, j = 10; i < j; i++, j--) {
    System.out.println(i + " " + j);
}

2.2 拡張for文（for-each）

配列やコレクションの全要素を順番に処理します。

拡張for文 Java 
// 配列の処理
int[] numbers = {1, 2, 3, 4, 5};
for (int num : numbers) {
    System.out.println(num);
}

// リストの処理
List<String> names = Arrays.asList("Alice", "Bob", "Charlie");
for (String name : names) {
    System.out.println(name);
}
⚠️ 拡張for文の制限

拡張for文ではインデックスを取得できません。また、ループ中に要素を削除するとConcurrentModificationExceptionが発生します。

Section 4-3

制御文の組み合わせ

3.1 制御文のネストと変数のスコープ

変数のスコープ Java 
for (int i = 0; i < 3; i++) {
    int temp = i * 2;  // ループ内でのみ有効
    System.out.println(temp);
}
// System.out.println(i);     // エラー：iはスコープ外
// System.out.println(temp);  // エラー：tempもスコープ外

3.2 繰り返し文のネスト

二重ループ（九九表） Java 
for (int i = 1; i <= 9; i++) {
    for (int j = 1; j <= 9; j++) {
        System.out.printf("%3d", i * j);
    }
    System.out.println();
}
Section 4-4

繰り返しの制御

4.1 無限ループ

無限ループの書き方 Java 
// while版
while (true) {
    // 処理
    if (条件) break;  // 終了条件でbreak
}

// for版
for (;;) {
    // 処理
}

4.2 break文

ループを即座に終了します。

break文 Java 
for (int i = 0; i < 10; i++) {
    if (i == 5) {
        break;  // i==5でループ終了
    }
    System.out.println(i);  // 0, 1, 2, 3, 4
}

4.3 continue文

現在の反復をスキップし、次の反復に進みます。

continue文 Java 
for (int i = 0; i < 5; i++) {
    if (i == 2) {
        continue;  // i==2をスキップ
    }
    System.out.println(i);  // 0, 1, 3, 4
}

4.4 ラベルによる制御

ネストしたループで外側のループを制御する場合に使用します。

ラベル付きbreak/continue Java 
outer:
for (int i = 0; i < 3; i++) {
    for (int j = 0; j < 3; j++) {
        if (i == 1 && j == 1) {
            break outer;  // 外側のループを終了
        }
        System.out.println(i + ", " + j);
    }
}
// 出力: 0,0  0,1  0,2  1,0
Exercise

4章 練習問題

Q1

以下のコードの出力回数は何回ですか？

int i = 10;
do {
    System.out.println(i);
    i++;
} while (i < 5);
  1. 0回
  2. 1回
  3. 5回
  4. 10回
Answer

4章 解説・解答

Q1の解答: B

do-while文は条件判定前に最低1回処理を実行します。iは10から始まり、条件(i < 5)はfalseですが、1回は出力されます。
