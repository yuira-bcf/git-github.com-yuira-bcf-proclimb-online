練習問題：レイアウトに関連する 10 題（解答例つき）

解答例は「必ずこの書き方でなければならない」という意味ではありません。
ただし、各問題の「制約」を満たすように、意図が伝わりやすい書き方で統一しています。

共通ルール（全問題）
  • 長さの単位はピクセル（px）を使用してください。
  • CSS は可能であればクラス名で指定してください（学習と管理のため）。
  • Flexbox / Grid の問題では、すき間の指定は gap を優先してください。

問1：Flexboxで横並びメニューを作成

目的：Flexbox による横並び、左右中央揃え、上下中央揃え、gap の使用

問題

ページ上部に、リンクが横一列に並ぶメニューを作成してください。

要件

  • HTML 構造：nav.menu の中に a 要素を 4 個入れてください。
  • .menu の高さは 56 ピクセルにしてください。
  • リンクは横一列、間隔は 16 ピクセルにしてください。
  • メニュー全体を左右中央、リンク文字を上下中央に見えるようにしてください。
  • リンクの下線を消し、文字色を黒にしてください。

制約

  • .menu に display: flex; を使用してください。
  • 間隔は gap を使用してください。
  • float、inline-block、text-align による中央寄せは禁止します。
解答例（HTML / CSS）を表示 
<!-- HTML -->
<nav class="menu">
  <a href="#">Home</a>
  <a href="#">News</a>
  <a href="#">About</a>
  <a href="#">Contact</a>
</nav>
/* CSS */
.menu{
  display: flex;
  justify-content: center; /* 左右方向の中央揃え */
  align-items: center;     /* 上下方向の中央揃え */
  gap: 16px;
  height: 56px;
}

.menu a{
  text-decoration: none;
  color: #000;
  padding: 6px 8px; /* クリックしやすいように余白を追加 */
}
ポイント：左右中央は justify-content、上下中央は align-items で指定します。

問2：Flexboxでカード一覧を作成し、折り返しを行う

目的：flex-wrap と gap、固定幅カードの一覧

問題

カードを複数並べ、画面幅が狭いときは次の行へ折り返してください。

要件

  • 親：div.card-list、子：div.card を 10 個
  • .card 幅 240px、内側余白 16px、すき間 16px
  • .card-list 最大幅 960px、左右中央

制約

  • display: flex; と flex-wrap: wrap;
  • すき間は gap
  • Grid は使用禁止
解答例（HTML / CSS）を表示 
<!-- HTML -->
<div class="card-list">
  <div class="card">Card 1</div>
  <div class="card">Card 2</div>
  <div class="card">Card 3</div>
  <div class="card">Card 4</div>
  <div class="card">Card 5</div>
  <div class="card">Card 6</div>
  <div class="card">Card 7</div>
  <div class="card">Card 8</div>
  <div class="card">Card 9</div>
  <div class="card">Card 10</div>
</div>
/* CSS */
.card-list{
  display: flex;
  flex-wrap: wrap;  /* 入りきらない場合は次の行へ */
  gap: 16px;
  max-width: 960px;
  margin: 0 auto;   /* 左右中央 */
}

.card{
  width: 240px;
  padding: 16px;
  border: 1px solid #e6e6e6;
  border-radius: 12px;
}
ポイント：折り返しは flex-wrap: wrap;、要素間のすき間は gap です。

問3：Gridで「ヘッダー・本文・サイドバー・フッター」を配置

目的：grid-template-areas によるページ骨組み

問題

ページの基本構成を Grid で作成してください。

要件

  • 親：div.layout
  • 子：header.header、main.main、aside.sidebar、footer.footer
  • 769px 以上：ヘッダー全幅 → 左 240px サイド / 右 本文 → フッター全幅
  • すき間 16px、最小高さ 100vh

制約

  • display: grid;
  • grid-template-areas を使用
  • position で配置しない
解答例（HTML / CSS）を表示 
<!-- HTML -->
<div class="layout">
  <header class="header">Header</header>
  <aside class="sidebar">Sidebar</aside>
  <main class="main">Main</main>
  <footer class="footer">Footer</footer>
</div>
/* CSS */
.layout{
  display: grid;
  grid-template-columns: 240px 1fr;
  grid-template-rows: auto 1fr auto;
  grid-template-areas:
    "header header"
    "sidebar main"
    "footer footer";
  gap: 16px;
  min-height: 100vh;
}

.header{ grid-area: header; }
.sidebar{ grid-area: sidebar; }
.main{ grid-area: main; }
.footer{ grid-area: footer; }

/* 見た目確認用（任意） */
.header, .sidebar, .main, .footer{
  border: 1px solid #e6e6e6;
  border-radius: 12px;
  padding: 12px;
}
ポイント：配置は「名前（areas）」で決めると読みやすく、修正もしやすいです。

問4：Gridで自動調整されるギャラリーを作成

目的：repeat(auto-fit, minmax()) による自動列数

問題

画面幅に応じて列数が自動で変わる一覧を作成してください。

要件

  • 親：div.gallery、子：div.item を 12 個
  • item 高さ 120px、すき間 12px
  • 列：最低 160px、余りは均等に広がる

制約

  • grid-template-columns: repeat(auto-fit, minmax(160px, 1fr)); を使用
  • @media 禁止
解答例（HTML / CSS）を表示 
<!-- HTML -->
<div class="gallery">
  <div class="item">1</div>
  <div class="item">2</div>
  <div class="item">3</div>
  <div class="item">4</div>
  <div class="item">5</div>
  <div class="item">6</div>
  <div class="item">7</div>
  <div class="item">8</div>
  <div class="item">9</div>
  <div class="item">10</div>
  <div class="item">11</div>
  <div class="item">12</div>
</div>
/* CSS */
.gallery{
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(160px, 1fr));
  gap: 12px;
}

.item{
  height: 120px;
  border: 1px solid #e6e6e6;
  border-radius: 12px;
  display: flex;
  justify-content: center;
  align-items: center;
}
ポイント：列数は固定せず、最小幅 160px を守りながら自動で増減させます。

問5：上下左右の中央配置を2通りで作成（Flexbox と Grid）

目的：中央配置を 2 つの方法で実装する

問題

親要素の中央に子要素を上下左右とも中央配置してください。実装方法は 2 通り示してください。

要件

  • .container：幅 400px、高さ 300px
  • .target：幅 120px、高さ 80px
  • 中央配置（上下左右）

制約

  • 方法1：Flexbox（display: flex）
  • 方法2：Grid（display: grid）
  • absolute + transform 禁止
解答例（HTML / CSS）を表示 
<!-- HTML（方法1：Flexbox） -->
<div class="container flex-center">
  <div class="target">Target</div>
</div>

<!-- HTML（方法2：Grid） -->
<div class="container grid-center">
  <div class="target">Target</div>
</div>
/* CSS（共通） */
.container{
  width: 400px;
  height: 300px;
  border: 1px solid #e6e6e6;
  border-radius: 12px;
}

.target{
  width: 120px;
  height: 80px;
  border: 1px solid #e6e6e6;
  border-radius: 12px;
  display: flex;
  justify-content: center;
  align-items: center;
}

/* 方法1：Flexbox */
.flex-center{
  display: flex;
  justify-content: center;
  align-items: center;
}

/* 方法2：Grid */
.grid-center{
  display: grid;
  place-items: center; /* 上下左右の中央 */
}
ポイント：Grid の place-items: center; は「上下左右の中央」を一行で表現できます。

問6：絶対位置指定で右上バッジを重ねる

目的：relative を基準に absolute で配置する

問題

カードの右上に「NEW」と書かれたバッジを重ねて表示してください。

要件

  • .card：幅 280px、内側余白 16px
  • .badge：上 8px、右 8px
  • はみ出し禁止

制約

  • .card に position: relative
  • .badge に position: absolute + top/right
  • float 禁止
解答例（HTML / CSS）を表示 
<!-- HTML -->
<div class="card">
  <span class="badge">NEW</span>
  <h4>カードのタイトル</h4>
  <p>カードの本文です。</p>
</div>
/* CSS */
.card{
  width: 280px;
  padding: 16px;
  border: 1px solid #e6e6e6;
  border-radius: 12px;
  position: relative; /* ここが基準になります */
}

.badge{
  position: absolute;
  top: 8px;
  right: 8px;
  font-size: 12px;
  padding: 4px 10px;
  border: 1px solid #e6e6e6;
  border-radius: 999px;
}
ポイント：absolute は「最も近い relative の親」を基準に配置されます。

動作イメージ（このページ内デモ）

問7：position: sticky で見出しを吸着させる

目的：sticky の基本（top / 背景 / z-index）と、効かない条件の理解

問題

スクロール中に見出しが画面上端へ吸着する見た目を作成してください。

要件

  • section を 3 つ以上
  • 各 section の先頭に h2.section-title
  • top: 0 で吸着
  • 背景色と z-index を指定

制約

  • position: sticky を使用
  • position: fixed 禁止
  • 追加課題：sticky が動作しない原因を 2 つ説明
解答例（HTML / CSS）を表示 
<!-- HTML -->
<section class="sec">
  <h2 class="section-title">第1章</h2>
  <p>...長い本文...</p>
</section>

<section class="sec">
  <h2 class="section-title">第2章</h2>
  <p>...長い本文...</p>
</section>

<section class="sec">
  <h2 class="section-title">第3章</h2>
  <p>...長い本文...</p>
</section>
/* CSS */
.section-title{
  position: sticky;
  top: 0;               /* 吸着位置を指定 */
  background: #fff;     /* 本文の上に重なっても読めるように */
  z-index: 1;           /* 本文より前面に */
  margin: 0;
  padding: 10px 8px;
  border-bottom: 1px solid #e6e6e6;
}

/* 追加課題（文章例）
原因1：top を指定していない（sticky の吸着位置が決まらない）
原因2：親要素に overflow: hidden / auto などがあり、sticky の基準やスクロール範囲に影響する場合がある
*/
ポイント：sticky は「固定（fixed）」ではなく「ある位置に来たら吸着」する挙動です。必ず top を指定します。

問8：float で文章の回り込みを作成し、解除する

目的：float の回り込みと clear による解除

問題

画像の代わりの箱の周囲に文章が回り込む見た目を作り、次の見出しでは回り込みを解除してください。

要件

  • .figure：幅 160px、高さ 120px、右と下の外側余白 16px
  • figure を左に寄せ、文章を回り込ませる
  • next-title は回り込まず下から表示

制約

  • float: left を使用
  • 解除に clear: both を使用
  • Flexbox / Grid で回り込み再現は禁止
解答例（HTML / CSS）を表示 
<!-- HTML -->
<div class="article">
  <div class="figure">図</div>
  <p>
    ここに長い文章を入れます。図の右側に文章が回り込みます。
    回り込みが分かるように、ある程度の長さにしてください。
  </p>
</div>

<h2 class="next-title">次の見出し（回り込み解除）</h2>
/* CSS */
.figure{
  float: left;
  width: 160px;
  height: 120px;
  margin: 0 16px 16px 0; /* 右と下に余白 */
  border: 1px solid #e6e6e6;
  border-radius: 12px;
}

.next-title{
  clear: both; /* 回り込み解除 */
}
ポイント：回り込みは float、解除は clear です。目的がはっきりしているのが float の良い点です。

問9：メディアクエリで2列→1列に切り替える（Gridで実装）

目的：Grid と @media によるレスポンシブ切り替え

問題

画面幅によって 2 列レイアウトと 1 列レイアウトを切り替えてください。

要件

  • 769px 以上：2列（左 240px サイド、右 本文）
  • 768px 以下：1列（header → main → aside → footer）
  • すき間 16px

制約

  • @media(max-width: 768px) を使用
  • HTML の並び順は変更しない
  • ページ骨格を Flexbox だけで作らない（Grid の練習）
解答例（HTML / CSS）を表示 
<!-- HTML -->
<div class="layout">
  <header class="header">Header</header>
  <main class="main">Main</main>
  <aside class="sidebar">Sidebar</aside>
  <footer class="footer">Footer</footer>
</div>
/* CSS（769px以上の基本形：2列） */
.layout{
  display: grid;
  grid-template-columns: 240px 1fr;
  grid-template-rows: auto 1fr auto;
  grid-template-areas:
    "header header"
    "sidebar main"
    "footer footer";
  gap: 16px;
  min-height: 100vh;
}

.header{ grid-area: header; }
.main{ grid-area: main; }
.sidebar{ grid-area: sidebar; }
.footer{ grid-area: footer; }

/* CSS（768px以下：1列に切り替え） */
@media (max-width: 768px){
  .layout{
    grid-template-columns: 1fr;
    grid-template-rows: auto auto auto auto;
    grid-template-areas:
      "header"
      "main"
      "sidebar"
      "footer";
  }
}
ポイント：HTML の順序は変えず、CSS の grid-template-areas を切り替えて見た目の並びを変えます。

問10：コンテナクエリで部品の幅に応じてレイアウトを変える

目的：画面幅ではなく「親要素の幅」でレイアウトを切り替える

問題

同じカード部品を、置かれた場所（親要素）の幅に応じて横並び・縦並びに切り替えてください。

要件

  • 親：wrap-wide（幅 600px）と wrap-narrow（幅 320px）を用意
  • 各親の中に同じ profile-card（photo + text）を置く
  • 親の幅が 500px 以上：横並び（photo 幅 160px）
  • 親の幅が 499px 以下：縦並び（photo 幅 100%）

制約

  • container-type と @container を使用
  • @media 禁止
  • JavaScript 禁止
解答例（HTML / CSS）を表示 
<!-- HTML -->
<div class="wrap-wide">
  <div class="profile-card">
    <div class="photo">Photo</div>
    <div class="text">
      <strong>氏名</strong>
      <p>紹介文が入ります。</p>
    </div>
  </div>
</div>

<div class="wrap-narrow">
  <div class="profile-card">
    <div class="photo">Photo</div>
    <div class="text">
      <strong>氏名</strong>
      <p>紹介文が入ります。</p>
    </div>
  </div>
</div>
/* CSS */
.wrap-wide{ width: 600px; border: 1px solid #e6e6e6; border-radius: 12px; padding: 12px; }
.wrap-narrow{ width: 320px; border: 1px solid #e6e6e6; border-radius: 12px; padding: 12px; margin-top: 12px; }

/* 親要素（切り替え基準）をコンテナとして宣言します */
.wrap-wide, .wrap-narrow{
  container-type: inline-size;
}

/* profile-card の基本（まずは縦並びにしておくと切り替えが理解しやすいです） */
.profile-card{
  display: flex;
  flex-direction: column;
  gap: 12px;
  border: 1px solid #e6e6e6;
  border-radius: 12px;
  padding: 12px;
}

.photo{
  width: 100%;
  height: 120px;
  border: 1px solid #e6e6e6;
  border-radius: 12px;
  display: flex;
  justify-content: center;
  align-items: center;
}

/* 親要素の幅が 500px 以上のときだけ横並びに変更します */
@container (min-width: 500px){
  .profile-card{
    flex-direction: row;
    align-items: center;
  }
  .photo{
    width: 160px;
    flex: 0 0 160px; /* 160px を固定 */
  }
}
ポイント：切り替えの条件は「画面幅」ではなく「親要素の幅」です。そのため、@media ではなく @container を使います。
